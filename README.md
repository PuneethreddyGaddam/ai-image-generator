# AI Image Generator

## Project Description
AI Image Generator is a simple web application that generates images using AI based on user text input.  
This project is built using **Node.js** and **Express** for the backend, and **HTML, CSS, and JavaScript** for the frontend.  
It uses the **OpenAI API** to generate images.

The main purpose of this project is to learn how to integrate AI APIs with a full-stack web application.

---
## Features
- Generate AI images from text prompts
- Simple and user-friendly interface
- Backend API using Node.js and Express
- Frontend built with HTML and CSS
- Uses OpenAI API for image generation
- API key protected using environment variables

---
## Prerequisites
Before running this project, you need:
- Node.js installed on your system
- npm (Node Package Manager)
- An OpenAI API key

---
## Installation

### Backend Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/PuneethreddyGaddam/ai-image-generator.git
   Go to the project folder:

cd ai-image-generator


2. Install dependencies:

npm install


3. Create a .env file in the root folder and add your API key:
   -OPENAI_API_KEY=your_api_key_here


4.Start the server:
-npm start

# Frontend Setup
- Open the public folder.
- Open index.html in your browser
- OR
- Run the backend and open:
- http://localhost:5000

# Usage
- Open the application in your browser.
- Type a text prompt (example: A futuristic city at night).
- Click the Generate button.
- Wait a few seconds for the AI image to generate.
- The image will be displayed on the screen.
